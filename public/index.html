Dashboard

Stats and charts use filtered hunts. Cumulative line merges encounters. Rate chart draws one line per session (active = solid, past = dashed) plus a mean line when multiple sessions exist. Hover points for details. Shiny sprites list every target on every active filtered hunt.

Encounter rate by session

Each point is net encounter change in a fixed wall-clock window (20 minutes by default), scaled to enc/h — not the gap between individual clicks. Segments outside mean ± 5σ are dropped iteratively. Each line is one hunt; past hunts are dashed. The lighter line is a mean across sessions. Hover the dots for hunt name, rate, and time.
